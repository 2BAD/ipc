Client SDK Instructions

NET_SDK_FindNextFile

get file information one by one

 LONG NET_SDK_FindNextFile(
 LONG                  lFindHandle,
 NET_SDK_REC_FILE      lpFindData
 );

Parameters

lFindHandle
[in] handle of finding file,return value of NET_SDK_FindFile()
lpFindData
[out] pointer to store file information

Return Values

-1 means failure and other value is current state information. To get error information,please call NET_SDK_GetLastError

Remarks

Before calling this interface,call NET_SDK_FindFile to get current handle for finding
