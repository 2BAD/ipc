Client SDK Instructions

NET_SDK_RefreshPlay

refresh window to display playback

 BOOL NET_SDK_RefreshPlay(
 LONG    lPlayHandle
 );

Parameters

lPlayHandle
[in] playback handle,return value of NET_SDK_PlayBackByTime

Return Values

TRUE means success; FALSE means failure. To get error information,please call NET_SDK_GetLastError

Remarks

When pause or playback in one frame,if refresh the window,the image disappears ,at that time call this interface to display,this interface is valid just when pause and playback in one frame.
