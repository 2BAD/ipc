Client SDK Instructions

DD_LIVE_VIDEO_GROUP

struct of preview video group

 struct _dd_live_video_group_{
 unsigned short	holdTime;	
 unsigned short	channelNum;
 unsigned long	       splitMode;	
 unsigned char	       channel [DD_MAX_CAMERA_NUM]; 
 }DD_LIVE_VIDEO_GROUP;

Members

holdTime
hold time,(in second), 0 means invalid
channelNum
valid channel number(read only)
splitMode
split mode, refer to DD_VIEW_SPLIT_MODE:
type Value Description
DD_VIEW_SPLIT_1X1 0x00010000 1X1
DD_VIEW_SPLIT_2X2 0x00020000 2X2
DD_VIEW_SPLIT_1A2/ DD_VIEW_SPLIT_2X3 0x00040000 2X3
DD_VIEW_SPLIT_1A5/DD_VIEW_SPLIT_3X3 0x00080000 3X3
DD_VIEW_SPLIT_1A7/DD_VIEW_SPLIT_1A12/DD_VIEW_SPLIT_4X4 0x00100000 4X4
DD_VIEW_SPLIT_2A6/DD_VIEW_SPLIT_4X6 0x00200000 4X6
DD_VIEW_SPLIT_1A9/DD_VIEW_SPLIT_4A9/DD_VIEW_SPLIT_1A16/DD_VIEW_SPLIT_4A16/DD_VIEW_SPLIT_5X5 0x00400000 5X5
DD_VIEW_SPLIT_1A11/DD_VIEW_SPLIT_1A20/DD_VIEW_SPLIT_4A20/DD_VIEW_SPLIT_6X6 0x00800000 6X6
channel [DD_MAX_CAMERA_NUM]
channel number corresponding to each area,array index stands for channel number,element value stands for window area number,0xff means invalid channel
