Client SDK Instructions

PTZ_3D_POINT_INFO

information about PTZ 3D control

 struct PTZ_3D_POINT_INFO{
 int      selBeginX;		
 int      selBeginY;	
 int      selEndX;		
 int      selEndY;	
 int      displayWidth;		
 int      displayHeight;	
 int      reserve[2];	
 }PTZ_3D_POINT_INFO;

Members

selBeginX
X coordinates of the starting point
selBeginY
Y coordinates of the starting point
selEndX
End point X coordinates
selEndY
End point Y coordinates
displayWidth
Image width
displayHeight
Image height
reserve[2]
Retention value,not enable

Remarks

4 coordinate variable is the mouse position relative to the current window of the upper left corner of the screen. enlarge:selBeginX < selEndX, narrow:selBeginX > selEndX
