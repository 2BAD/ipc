Client SDK Instructions

NET_SDK_CLIENTINFO

struct of log information

struct _net_sdk_clientinfo{ 
LONG lChannel; 
LONG streamType; 
HWND hPlayWnd; 
int bNoDecode;
}NET_SDK_CLIENTINFO, *LPNET_SDK_CLIENTINFO;

Members

lChannel
channel number,start from 0
streamType
data stream type,two types:NET_SDK_MAIN_STREAM and NET_SDK_SUB_STREAM
hPlayWnd
play window handle
bNoDecode
0:decode,1:not decode.only for windows os,default value is 0
