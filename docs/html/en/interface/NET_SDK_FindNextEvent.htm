NET_SDK_FindNextEvent

get information of found file one by one

 LONG NET_SDK_FindNextEvent(
 LONG                     lFindHandle,
 NET_SDK_REC_EVENT        *lpRecEvent
 );

Parameters

lFindHandle
[in] handle of finding file,return value of NET_SDK_FindEvent()
*lpRecEvent
[out] pointer to store file information

Return Values

-1 means failure and other value is current state information. To get error information,please call NET_SDK_GetLastError

Remarks

Before calling this interface,call NET_SDK_FindEvent to get current handle for finding.The record file found by event aims at the start time and stop time,so only supports playback by time.
