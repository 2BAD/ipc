Client SDK Instructions

NET_SDK_StartVoiceCom_MR

start voice forward function

 LONG NET_SDK_StartVoiceCom_MR( 
 LONG                   lUserID,
 TALK_DATA_CALLBACK     fVoiceDataCallBack,
 void                   * pUser
 );

Parameters

lUserID
[in] user ID
bNeedNoEncodeData
[in]  whether to encode the language data. If TRUE is selected, the stream from the device will be directly called back and the stream transfered to the device is encoded by SDK . If FALSE is selected, the stream from the deivce is decoded first and then called back; the stream transfered to the device is not encoded by SDK.
fVoiceDataCallBack
[in] return value of voice data callback function
pUser
[in] user information

Return Values

-1 means failure and other value is return value of voice forward. To get error information,please call NET_SDK_GetLastError
