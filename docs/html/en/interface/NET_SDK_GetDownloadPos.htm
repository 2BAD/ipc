Client SDK Instructions

NET_SDK_GetDownloadPos

get current progress of downloading record file

 int NET_SDK_GetDownloadPos(
 LONG    lFileHandle
 );

Parameters

lFileHandle
[in] handle for downloading,return value of NET_SDK_GetFileByTime()

Return Values

-1 means failure;0-100 means the progress of downloading;100 means finish;normal range is 0-100, if the return value is 200,network is unusual. To get error information,please call NET_SDK_GetLastError

Remarks

This interface is to get progress of downloading record file by file name.
