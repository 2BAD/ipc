Client SDK Instructions

NET_SDK_LivePlay

real time preview

 LONG NET_SDK_LivePlay(
 LONG                     lUserID,
 LPNET_SDK_CLIENTINFO     lpClientInfo,
 LIVE_DATA_CALLBACK       fLiveDataCallBack, 
 void                     * pUser
 );

Parameters

lUserID
[in] return value of NET_SDK_Login()
lpClientInfo
[in] preview parameter
fLiveDataCallBack
[in] preview data callback parameter,default value is NULL
* pUser
[in] pointer to user,default value is NULL

Return Values

-1 means failure and other value is parameter of handle of function NET_SDK_StopLivePlay. To get error information,please call NET_SDK_GetLastError

Remarks

After calling this interface successfully,if need to capture real time code stream data,call NET_SDK_SetLiveDataCallBack to register the callback function for capturing code stream data,and access the code stream data in the callback function.
