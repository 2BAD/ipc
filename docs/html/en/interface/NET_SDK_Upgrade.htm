Client SDK Instructions

NET_SDK_Upgrade

remote upgrade

 LONG NET_SDK_Upgrade(
 LONG    lUserID,
 char   *sFileName
 );

Parameters

lUserID
[in] return value of NET_SDK_Login
wVolume
[in] directory of file upgrade(include file name)

Return Values

-1 means failure and other values are parameter of NET_SDK_GetUpgradeState. To get error information,please call NET_SDK_GetLastError
