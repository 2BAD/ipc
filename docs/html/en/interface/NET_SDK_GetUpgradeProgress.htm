NET_SDK_GetUpgradeProgress

get progress of remote upgrading

 int NET_SDK_GetUpgradeProgress(
 LONG   lUpgradeHandle
 );

Parameters

lUpgradeHandle
[in] return value of NET_SDK_Upgrade

Return Values

-1 means failure and 0-100 means progress of upgrading. To get error information,please call NET_SDK_GetLastError
