NET_SDK_FindNextLog

get log information one by one(not support IPC)

 LONG NET_SDK_FindNextLog(
 LONG                lLogHandle,
 LPNET_SDK_LOG       lpLogData
 );

Parameters

lLogHandle
[in] handle for finding log.return value of NET_SDK_FindDVRLog()
lpLogData
[out] pointer to store log information

Return Values

-1 means failure and other values are current state information. To get error information, please call NET_SDK_GetLastError

Remarks

Before calling this interface,call NET_SDK_FindDVRLog to get current handle for finding.
